# sqlit

A browser-based SQLite notebook. Think Jupyter, but with native SQL support and database interop with Python through the `sqlite3` module. Runs entirely client-side.

## Setup

### Prerequisites

- Node.js + pnpm

### Install dependencies

```bash
pnpm install
```

### Start the dev server

```bash
pnpm dev
```

## Commands

```bash
pnpm dev      # Start dev server
pnpm build    # Type-check and build for production
pnpm lint     # Run Biome linter
pnpm format   # Format code with Biome
pnpm preview  # Preview production build
```
